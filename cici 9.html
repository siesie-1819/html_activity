Reviewer in web Development

HTML Editors
A computer program used for editing HTML, which is the markup of a web page.
*Notepad, Wordpad.
Text Editor
A simple, light program used to write, open, view, and edit plain text documents. It does not use styles or formatting.
* Notepad++.
WYSIWYG Editors
An editor where you can compose your web page exactly the way it looks in a browser. The acronym stands for "What You See Is What You Get".
*Adobe Dreamweaver.
[!DOCTYPE html]
A declaration that informs the web browser that the document being rendered is an HTML document. It is not an HTML tag but is required at the beginning of every HTML document.
*[!DOCTYPE HTML] followed by the [html] tag.
[html] Tag
The tag used after the [!DOCTYPE] declaration to indicate to the browser that this is an HTML document. All HTML files must begin and end with this tag.
*[html]...[/html].
[body] Tag
This section contains everything that will be displayed on your web page, including text, images, and videos.
*[body]...your page content...[/body].
[title] Tag
This tag, located inside the [head] section, is used to specify the title of the web page that is displayed on the title bar of the browser.
*[title>My First Web Page[/title].
Heading Tags ([h1] to [h6])
Container tags that format text using predefined values for size and color, commonly used for titles and headers. The higher the number, the smaller the font size.
[h1]*This is a main heading[/h1] or [h6]This is a subheading[/h6].
Paragraph Tag ([p])
A tag that defines a paragraph. It automatically creates a new line after each declaration.
*[p]This is a paragraph.[/p].
Meta tags
Describe metadata (a set of data that describes and gives information about other data.) within an HTML document.
* [meta charset="UTF-8"]
link tags
It connects a document to an external document.
* [link rel="stylesheet" type="text/css" href="theme.css"]
Style tags
The [style] tag is used to define style information for an HTML document.
*[style]h1
{color:red;}
p {color:blue;}
[/style]
Line Break
The Line Break [br] tag creates a line break or a new line
* [p]I am[br]a student
Horizontal Line
It allows you to underline headlines or complete lines of text
* [hr color="white" width="20000" size="4" align="center"]
Blockquote
The HTML [blockquote] element defines a section that is quoted from another source.
* [blockquote cite="http://www.worldwildlife.org/who/index.html"]
Invinsible Comment
Used to put significant statements and/or remarks that you do not want to be displayed in the browser.
* [!- - this is my comment and this will not be displayed in the browser - -]
Background Color
To give design to your background
* [h1 style="background-color:DodgerBlue;"]Hello World[/h1]
Text Color
To give design to your text
* [h1 style="color:Tomato;"]Hello World[/h1]
